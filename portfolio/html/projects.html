Real Estate Listing Website

Express JS website allowing users to buy and sell real estate properties

Github Repo

About this project

The project was developed by a team of 3 students to create a real-estate listing website. The websites main features allow registered users to sell properties and also search and view detailed description of nearby properties for sale through a search engine. My role in the project was to create a sell page which allowed for storing of listing data in the database and displaying the listing details from the database when a listing is viewed.

Tools

On the technical side, we used MySQL as our database to store data, nginx to serve static files and express primarily for its middleware features like cookie parsing, routing and web developnment capabilities. To display HTML content, we used EJS as it allows for easy injection of JavaScript, thus, easy implementation of dynamic and responsive UI.

Challenges

The first challenge I faced as a developer was learning a new language Express JS from scratch in a short period of time. The second challenge I faced was ensuring my code followed a similar coding structure as my teammates.

Solutions

In performing my responsibilities, I had to first thoroughly teach myself about the Express framework. I was able to accomplish this through reading documentations, watching videos as resource and receiving help from my teammates. We would frequently share with each other documentations to further our understanding of the language. While coding, we would consult with each other to finds ways to better approach implementing a feature as well to ensure we are all following similar coding and a organized project file structure for better clarity when reviewing each others code.

Reflection

Through my experience developing the website in a team of three, I was able to strengthen my ability to read and produce good documentation so that others can view with ease. Additionally, I was able to strengthen my HTML, CSS and Express JS knowledge.
